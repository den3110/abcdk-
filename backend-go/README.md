# backend-go

Go-first backend cutover runtime.

Commands:

- `go run ./cmd/api`
- `go run ./cmd/relay-rtmp`
- `go run ./cmd/scheduler`
- `go run ./cmd/worker-recording-export`
- `go run ./cmd/worker-ai-commentary`
- `go run ./cmd/worker-general`

The repo root `go run .` remains an alias for `cmd/api` during transition.
